# Contributing to VORTEX System

Thank you for your interest in this project! VORTEX is currently an early-stage conceptual exploration. All contributions — big or small — are welcome.

---

This repository currently focuses on the VORTEX system, with more conceptual projects (such as honeycomb arcology designs) coming soon. The same contribution spirit applies across all ideas.


## Ways to Contribute

- **Feedback and discussion** — Questions, thoughts, alternative approaches, or real-world considerations are extremely valuable.
- **Ideas and improvements** — Suggestions for refinements, new variants, integration ideas, or related concepts.
- **Documentation** — Fixing typos, improving clarity, adding references, or expanding explanations.
- **Visuals** — Concept art, diagrams, or improved illustrations (descriptions or image files are both useful).
- **Research / Analysis** — Links to relevant papers, back-of-envelope calculations, CFD notes, or simulation ideas.
- **Adaptations** — Feel free to fork the repo and take the concepts in new directions.

## How to Contribute

1. Open an **Issue** or start a **Discussion** on GitHub — this is the best place to share thoughts right now.
2. For direct changes, fork the repository, make your edits, and open a Pull Request.
3. Simple typo fixes or clarifications can be submitted directly via Pull Request.

There is no formal review process yet — contributions will be reviewed as time allows. The goal is collaborative inspiration rather than production code.

## Donations / Sponsorship (Optional)

If you find value in these concepts and would like to support further development (time, tools, or experiments), you can sponsor the project via [GitHub Sponsors](https://github.com/sponsors/RheaSnowden) (once set up) or other platforms. All support is greatly appreciated but never expected.
